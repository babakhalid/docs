# What Can I Build?

Explore the possibilities with Yomake. This guide showcases real-world applications, provides inspiration, and helps you understand what you can create.

---

## Overview

Yomake empowers you to build modern web applications using React, TypeScript, and Tailwind CSS. If it runs in a web browser, you can probably build it with Yomake.

**Technology stack:**
- **Frontend**: React + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Backend**: Supabase (optional)
- **Deployment**: Yomake or Vercel hosting

## What You Can't Build (Yet)

### Limitations

**Native mobile apps:**
- Yomake builds web apps, not iOS/Android apps
- Workaround: Build progressive web app (PWA)

**Backend-heavy logic:**
- Complex algorithms better suited for backend
- Workaround: Use Supabase Edge Functions

**3D graphics and games:**
- Game engines not supported
- Workaround: Simple canvas-based games possible

**Video streaming:**
- Live video encoding/streaming
- Workaround: Embed third-party players (YouTube, Vimeo)

**Native desktop apps:**
- Web apps only (browser-based)
- Workaround: Electron possible but manual setup

**Non-React frameworks:**
- Vue, Angular, Svelte not supported
- React ecosystem only

## Ready to Build?

**Pick a project and start:**

1. Choose something that solves a real problem
2. Start with a template close to your vision
3. Describe your idea to the AI
4. Build iteratively, one feature at a time
5. Deploy early, get feedback
6. Iterate based on usage

**Need help deciding?** Ask the AI:
```
I want to build [describe your idea]. What features should I include
and what's the best way to structure this project?
```

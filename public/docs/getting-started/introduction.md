## Limitations

**Yomake is best for:**
- Frontend web applications
- React-based projects
- Standard web technologies

**Yomake is not ideal for:**
- Native mobile apps (iOS/Android)
- Complex backend logic (use Supabase integration)
- 3D graphics or game development
- Non-React frameworks (Vue, Angular, etc.)

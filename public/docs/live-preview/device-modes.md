Test your application on different screen sizes using Yomake's built-in device preview modes.

---

## Overview

Device modes let you preview how your project looks and behaves on different devices without physically owning them. Test responsive designs, check mobile layouts, and ensure your application works beautifully on any screen size.

## Accessing Device Modes

### Device Toolbar

Located at the top of the preview panel. Choose from multiple device presets or create custom dimensions.

**Controls:**
- **Device Selector** - Choose device type
- **Zoom** - Adjust preview scale (25%-200%)
- **Rotate** - Switch between portrait and landscape
- **Fullscreen** - Expand preview to full screen

### Quick Keyboard Shortcuts

- \`Cmd/Ctrl + 1\` - Desktop view
- \`Cmd/Ctrl + 2\` - Tablet view
- \`Cmd/Ctrl + 3\` - Mobile view
- \`Cmd/Ctrl + R\` - Rotate device orientation
- \`Cmd/Ctrl + 0\` - Reset zoom to 100%

## Device Types

### Desktop (1920x1080)
- Standard desktop resolution
- Test full layouts and multi-column designs
- Verify navigation menus and sidebars

### Laptop (1366x768)
- Most common laptop resolution worldwide
- Test reduced horizontal space
- Ensure content fits without scroll

### Mobile - iPhone (375x667)
- Popular iPhone size
- Test mobile-first designs
- Verify touch targets (minimum 44px)

### Tablet - iPad (768x1024)
- Standard tablet size
- Test hybrid layouts
- Works for both orientations

For detailed information about responsive design best practices, see our [Quick Start Guide](/docs/getting-started/quick-start).

---

## Testing Workflow

**Recommended Order:**
1. Start with mobile (smallest screen)
2. Test tablet in both orientations
3. Check laptop view
4. Verify desktop layout
5. Test custom breakpoints
